# laconic2d

Install and run `laconic2d`:

  ```bash
  # install the laconic2d binary
  make install

  # initialize the chain
  make init

  # start the chain
  laconic2d start --gql-playground --gql-server
  ```

Run tests:

  ```bash
  # integration tests
  make test-integration

  # e2e tests
  make test-e2e
  ```
